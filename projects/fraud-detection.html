Real-time Fraud Detection System

Problem Statement

Financial institutions face significant challenges in detecting fraudulent transactions in real-time. Traditional batch processing systems often identify fraud after the fact, resulting in substantial financial losses. This project aims to develop a real-time fraud detection system capable of identifying suspicious transactions as they occur.

Technology Stack

Apache Kafka Apache Spark Streaming Python scikit-learn Docker AWS
